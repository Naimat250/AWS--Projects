Demo AWS CI-CD Pipeline

Version: 02 - Auto Build & Deploy

Project Workflow

This workflow demonstrates the AWS CI/CD process triggered by a GitHub commit.

Step 1: Code Push to GitHub

Developer pushes the latest code changes to the GitHub repository.

Step 2: AWS CodePipeline Triggered

AWS CodePipeline detects the commit and automatically starts the build process.

Step 3: Build & Deploy

AWS CodeBuild compiles the code, and CodeDeploy deploys the new version to the environment.
